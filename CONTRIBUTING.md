# 贡献指南

感谢你对 RED Skill 矩阵项目的关注！

## 如何贡献

### 提交新 Skill
1. Fork 本仓库
2. 在 `skills/` 下创建新目录
3. 编写 `SKILL.md`（参考[模板](./skills/_template/)）
4. 确保通过内容合规检查
5. 提交 PR

### Skill 命名规范
- 目录名：英文小写 + 连字符（如 `title-generator`）
- SKILL.md 中 `name` 字段：中文名称
- 遵循[平台规则](./docs/platform-rules.md)

### 提交规范
- Commit message: `<type>: <description>`
- Type: feat / fix / docs / chore
- 示例：`feat: 添加爆款标题生成器 v0.1`

## 行为准则
- 原创内容，不抄袭
- 遵守小红书平台规范
- 友善协作，尊重他人
