### 模式A 工作流

#### Step 1：赛道扫描

用网络搜索采集赛道信息：

**搜索策略：**
- "小红书 [赛道] 博主推荐 2026"
- "小红书 [赛道] 爆款账号 分析"
- "[赛道] 小红书 运营 心得"
- "小红书 [赛道] 变现"
- "[赛道] 自媒体 定位方法"

**采集目标：**
- 赛道里有哪几种博主类型
- 哪种内容方向最火
- 主要变现路径
- 竞争程度（饱和/中等/蓝海）
- 内容空白点

#### Step 2：引导找对标

给用户精确的搜索清单 + 模板，让用户在小红书找到对标账号。

**对标账号筛选4标准：**
- 粉丝 1000-10000（起步期，路径可学）
- 笔记 ≤ 100篇（还没太成熟）
- 有 ≥ 3篇千赞爆文（证明内容有效）
- 最近30天还在更新（活跃）

#### Step 3：对标拆解（用户带数据回来后）

对每个对标账号做6维拆解：

| 分析维度 | 拆什么 |
|----------|--------|
| 昵称分析 | 命名规律：身份+领域 / 趣味+领域 / 真名+领域 |
| 简介分析 | 结构：身份→价值→频率→联系方式 |
| 内容定位 | 主打什么类型？比例怎么分？ |
| 人设标签 | 什么身份？什么语气？什么记忆点？ |
| 爆文共性 | 爆文标题结构、封面风格、内容类型 |
| 运营节奏 | 发布频率、发布时间、系列化程度 |

**输出共性提炼：**
- 3-5个对标的共同优点 → 赛道基本盘，必须做到
- 差异化机会 → 他们都没做好但你可以做的

#### Step 4：市场调研

| 调研维度 | 分析什么 |
|----------|---------|
| 目标受众 | 谁在看？年龄/性别/需求/消费力 |
| 内容供给 | 哪种多？哪种少？空白在哪？ |
| 竞争格局 | 头部有谁？腰部有谁？你的位置？ |
| 变现潜力 | 赛道变现路径 + 你适合走哪条 |

#### Step 5：输出包装方案（见下方统一格式）

## 分析方法

**对标筛选4标准**：粉丝1000-10000 + 笔记≤100篇 + 赞藏评均衡 + 千赞爆文≥3篇

**昵称命名3公式**：
1. 身份+领域：「职场老学姐」「HR小课堂」→ 搜索友好
2. 趣味+领域：「打工日记」「搬砖研究所」→ 有记忆点
3. 真名/昵称+领域：「小贝的职场笔记」→ 信任感强

**简介万能结构**：身份标签 → 价值承诺 → 更新频率 →（可选）联系方式
> 例：「95后HR｜帮你看穿职场套路｜每周3更｜职场问题私信」

**人设3要素**：语气（怎么说话）+ 视觉（怎么呈现）+ 记忆点（怎么被记住）

**差异化寻找法**：列出对标的共同弱点（更新慢/回复少/没系列化/封面不统一/没变现），选1-2个你能做到更好的，那就是你的差异化。

## 红线

- 不要求AI自动浏览小红书找账号（AI做不到，引导用户自己搜）
- 不抄袭对标账号的昵称/简介（参考结构，不照搬内容）
- 不涉及医疗/金融承诺
- 昵称/简介不含违禁词（最/第一/国家级等极限词）
- 不引导站外引流（简介不放微信号/手机号，小红书会限流）
- 禁用词替换：最→保姆级 | 第一→首选 | 100%→几乎全部
