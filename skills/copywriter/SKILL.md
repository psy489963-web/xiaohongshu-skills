# 小红书文案优化器

把一段普通文案改出爆款体质。不改你的原意，只补齐4个致命缺位：钩子/结构/细节/互动。

## 安装

```bash
openclaw skills install @psy489963/copywriter
```

## ⚡ 闪电开始

```
用户：帮我优化这篇小红书笔记 ↓
"最近换了新的洗面奶，感觉还不错，泡沫很丰富，洗完不紧绷，推荐给大家试试"
```

你输出：

| 问题 | 说明 |
|------|------|
| 缺钩子 | 开头平铺直叙，3秒内留不住人 |
| 缺结构 | 一段到底，没有分段和节奏 |
| 缺细节 | "感觉还不错"太模糊，没有具体场景 |
| 缺互动 | 结尾没有引导评论/收藏 |

**我改了什么：** 加了痛点钩子→分段+emoji→3个具体变化→60块/3个月的价格锚点→开放式提问互动。

"改了什么"一目了然，用户能学会自己下次怎么写。下面进入正题。

## 工作流

### Step 1 — 诊断（4维扫描）

读完用户正文，按以下4维逐一诊断，只说真有问题的：

| 维度 | 检查什么 | 没问题的标志 |
|------|---------|-------------|
| **钩子** | 前2行有痛点/悬念/反常识吗？ | 让人想接着往下看 |
| **结构** | 有分段/小标题/节奏感吗？ | 不密集，一眼能看到重点 |
| **细节** | 有具体数字/品牌/价格/场景吗？ | 读起来像真人体验，不像说明书 |
| **互动** | 结尾有开放式提问/投票/收藏引导吗？ | 让人想说点什么 |

诊断结果用表格输出，问题维度标🔴，合格标🟢，一句话说清问题在哪。

### Step 2 — 匹配结构

根据用户笔记类型，从下方10种公式中推荐1-2个最合适的：

**注意力钩子型**
- AIDA结构：Attention(抓眼球)→Interest(引发兴趣)→Desire(激发欲望)→Action(引导行动)。适合有转化目标的笔记。
- 突破常识：说常识→颠覆反转→论证→正确指南。适合有"刻板印象"可打破的赛道。

**价值输出型**
- 干货教程：价值承诺→框架地图→分步详解→工具分享。适合知识/技能类。
- 资源合集：精准主题→筛选标准→清单罗列→分类推荐。适合好物/书单/探店。
- 痛点解决：场景化痛点→自身共情→分点方案→效果验证。适合职场/育儿/护肤。

**信任构建型**
- 真诚利他：身份+成果→公开干货→细节补充→深度连接。适合知识付费/育儿。
- 测评对比：背景声明→维度公开→结果可视化→场景推荐。适合美妆/数码。
- 避坑预警：踩坑故事→坑点总结→行动指南→转发号召。适合消费/旅行。

**传播裂变型**
- 情绪共鸣：核心情绪→细节场景→价值升华→互动引导。适合情感/成长。
- 争议讨论：抛出议题→观点拆解→个人态度→引导辩论。适合有对立观点的赛道。

### Step 3 — 改写执行

基于诊断+匹配的结构，改写用户原文。

**改写铁律：**
1. 不改原意，只补缺位
2. 开头必须有钩子（痛点/悬念/反常识/数据，四选一）
3. 每120字至少1个具象细节（价格/品牌/时间/地点/数量）
4. 禁用词：绝了、封神、天花板、必入、王炸、私信我、加微信
5. 自然嵌入2-3个关键词（方便搜索），不堆砌
6. emoji适量，用在段落标记和强调处，不泛滥

**输出格式：**
```
[优化版正文，直接可复制发布]

---
📊 优化报告
- 结构：用了什么公式
- 我改了什么：具体改动清单（钩子/分段/细节/互动各做了什么）
- 字数：优化前⌛→优化后⌛
- 建议搭配：什么风格的封面图
```

### Step 4 — 合规检查

逐条过：
- [ ] 无禁用词（绝了/封神/天花板/必入/王炸/私信/微信/加群/全套资料）
- [ ] 无过度承诺（"三天""立竿见影""百分百"）
- [ ] 标签 ≤ 10个，不挂无关话题
- [ ] 结尾有自然互动引导（开放式提问，不硬要赞藏评）
- [ ] 内容与标题一致，不挂羊头卖狗肉

---

## 发布检查清单

优化完别忘了：
1. 标题有力吗？→ 用标题生成器再跑一遍
2. 封面吸睛吗？→ 用对比/清单/场景型封面模板
3. 话题标签加了吗？→ 5-10个，大小标签搭配
4. 发完后回评论区了吗？→ 前1小时回复每条评论，助推互动率

---

## 多版本输出

用户说"再给我一版"时，换一种结构公式重写。比如：
- 第一版用AIDA → 适合转化
- 第二版用情绪共鸣 → 适合传播
- 第三版用痛点解决 → 适合收藏

每版标注用了什么公式、为什么适合。让用户有选择权。

## 5种文案类型速查

| 类型 | 开头钩子 | 核心节奏 | 结尾引导 |
|------|---------|---------|---------|
| 种草推荐 | 痛点共鸣"之前总...直到..." | 外观→使用感→对比→价格 | "还有什么想了解的？" |
| 干货教程 | 问题+数据"你是不是也..." | 步骤清单①→②→③带案例 | "收藏起来慢慢试❤️" |
| 故事叙述 | 悬念/冲突"这不是爽文..." | 时间锚点→波折→转折 | "你们遇到过吗？" |
| 好物清单 | 人群+场景锁定 | 每款：名字+亮点+价格+适合谁 | "你最爱哪一款？" |
| 避坑指南 | 踩坑故事+情绪 | 坑点①→②→③ + 正确做法 | "你还踩过什么坑？" |
