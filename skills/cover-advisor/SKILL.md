## 工作流（3步）

### Step 1：诊断 — 分析你的内容，推荐最佳封面风格

根据你提供的内容信息，我判断：

| 分析维度 | 具体判断 |
|----------|----------|
| 内容类型 | 你笔记属于哪类？（教程/经验/测评/合集/感悟…） |
| 点击驱动 | 什么会让用户停下来？（数字震撼/对比可信度/痛点共鸣/好奇…） |
| 推荐风格 | 从6种爆款风格中推荐1-2种 |
| 推荐理由 | 为什么这个风格适合你 |

**6种爆款封面风格速查**：

| 风格 | 特征 | 最适内容 | 常用工具 |
|------|------|----------|----------|
| 杂志留白风 | 大留白、标题突出、≤2色、干净 | 穿搭/护肤/探店/旅行 | Canva / 稿定设计 |
| 质感氛围风 | 胶片感/颗粒感、真实感强、暖调 | 摄影/治愈/咖啡店/随手拍 | 醒图 / VSCO |
| 大色块冲击风 | 纯色满屏背景、超大标题、高对比 | 知识干货/教程/运营技巧 | Canva / 稿定设计 |
| 拼贴风 | 多图组合、结构清晰、信息量大 | 美食合集/旅行攻略/产品对比 | 醒图拼图 / 美图秀秀 |
| 手写小字风 | 柔和色调、仿真手写、真实感 | 情绪价值/心得分享/小贴士 | 黄油相机 / 醒图 |
| 对比图风 | 前后对比、左旧右新、箭头引导 | 减肥/改造/学习/护肤效果 | 醒图拼图 / Canva |

## 设计铁律（每个方案自动遵守）

1. **3色原则**：封面主要颜色不超过 3 种，主色+辅色+文字色
2. **文字可读**：手机屏上所有文字必须清晰可读，副标题 ≥ 字号 24
3. **信息锚点**：用户 0.3 秒就知道这笔记讲什么，不用猜
4. **工具可操作**：每个方案必须附带具体工具名+操作步骤，不给"试试配个色"这种模糊建议
5. **合规底线**：不用违禁词（最/第一/国家级等极限词），不用医疗断言（"治愈""根除"），不引导站外引流

## 进阶：配图排版建议

如果笔记需要多张配图（非封面），告诉我内容结构，我给出排版方案：

**常用配图排版模板**：

| 模板 | 结构 | 适合 |
|------|------|------|
| LIST 清单 | 1张封面 + N张步骤图，每张1个要点 | 教程/攻略/方法 |
