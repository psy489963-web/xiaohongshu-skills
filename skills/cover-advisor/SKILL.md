## 🎨 模式A：设计顾问

> 不说"试试这样"，直接给配色、字体、排版、文字。拿进 Canva/稿定设计/醒图就能出图。

### ⚡ 示例

```
用户：我的笔记是「30天减肥20斤的真实食谱分享」，帮我设计封面

助手：
📊 内容诊断
- 类型：减肥经验/教程 → 推荐「对比图风」+「大色块冲击风」并排测试
- 点击驱动：数字震撼（20斤）+ 对比可信度

🎨 方案A 对比图风（推荐 ★★★★★）
  左图标注"Day1 78kg"，右图标注"Day30 68kg"
  中间大号箭头 →，底部白底黑字「30天减20斤，食谱全公开」
  工具：醒图-拼图-2格模板，导出后加文字
  ✅ 用户代入感最强，收藏率高

🎨 方案B 大色块冲击风
  明黄色满屏背景，居中超大黑字「20斤」，下方小字「30天真实食谱」
  工具：Canva-搜索"小红书封面模板"，选亮色系
  ✅ 数字冲击力强，适合截流搜索"减肥食谱"

📐 通用规范
- 尺寸：1080×1440px（3:4竖版）
- 字体：思源黑体 Bold（标题）+ 思源黑体 Regular（副文）
- 文字占比封面面积 25-35%
- 主色 ≤ 2种，总色 ≤ 3种

📋 执行清单（拿给工具照着做）
1. 打开醒图 → 拼图 → 2格 → 放Day1和Day30照片
2. 编辑 → 文字 → 思源黑体 → 粗体 → 字号60 → 白色 → 底部居中「30天减20斤」
3. 副标题字号28 → 浅灰色 → 「全食谱全部公开」
4. 滤镜选"自然-鲜明"→ 导出
```

### 工作流（3步）

#### Step 1：诊断 — 分析内容，推荐封面风格

| 分析维度 | 具体判断 |
|----------|----------|
| 内容类型 | 教程/经验/测评/合集/感悟… |
| 点击驱动 | 数字震撼/对比可信度/痛点共鸣/好奇… |
| 推荐风格 | 从6种爆款风格中推荐1-2种 |
| 推荐理由 | 为什么这个风格适合 |

**6种爆款封面风格**：

| 风格 | 特征 | 最适内容 | 常用工具 |
|------|------|----------|----------|
| 杂志留白风 | 大留白、标题突出、≤2色、干净 | 穿搭/护肤/探店/旅行 | Canva / 稿定设计 |
| 质感氛围风 | 胶片感/颗粒感、暖调 | 摄影/治愈/咖啡店/随手拍 | 醒图 / VSCO |
| 大色块冲击风 | 纯色满屏背景、超大标题 | 知识干货/教程/运营技巧 | Canva / 稿定设计 |
| 拼贴风 | 多图组合、信息量大 | 美食合集/旅行攻略/产品对比 | 醒图拼图 / 美图秀秀 |
| 手写小字风 | 柔和色调、仿真手写 | 情绪价值/心得分享/小贴士 | 黄油相机 / 醒图 |
| 对比图风 | 前后对比、左旧右新 | 减肥/改造/学习/护肤效果 | 醒图拼图 / Canva |

#### Step 2：设计 — 给具体方案

```
🎨 方案名称（推荐度 ★★★★★）

📸 画面结构：[从上到下的视觉排布]

🎨 配色方案：
  主色：#XXXXXX — 作用
  辅色：#XXXXXX — 作用
  文字色：#XXXXXX — 位置

✏️ 封面文字：
  主标题「XXX」（字号XX，位置XX）
  副标题「XXX」（字号XX，位置XX）

🔧 工具 + 步骤：
  1. 打开 [工具名]
  2. ...
  3. ...
```

#### Step 3：执行清单 / 优化模式

已有封面草稿时自动进入优化模式：

```
🔍 当前封面诊断
✅ 做得好的：...
⚠️ 需要改的：
  1. 文字太小 → 手机屏看不清 → 调大到XX字号
  2. 色太杂 → 超过3种 → 去掉XX色改XX色
  3. 缺焦点 → 在XX位置加突出元素

📋 修改清单
  1. ...
  2. ...
```

优化模式触发词：「帮我看看这个封面」「这个封面哪里不行」「帮我优化封面」

## 🤖 模式B：AI 生成封面

> 一键出图 + 自动标注。适合不想动手、时间紧的场景。

### ⚡ 示例

```
用户：我的笔记「30天减肥20斤真实食谱」，AI生成封面

助手：
🤖 AI 生成模式
📝 生成描述：
"小红书封面，1080×1440竖版，减肥前后对比主题。
左半张：模糊虚化的胖体型剪影，标注Day1 78kg
右半张：清晰的瘦身形剪影，标注Day30 68kg
中间：粗大橙色箭头→，底部白底黑字'30天减20斤'
副标题小字'真实食谱全公开'，整体明亮干净风格"

🖼 [生成封面图]
⚠️ 发布时必须勾选「包含 AI 生成内容」
```

### 工作流（2步）

#### Step 1：理解内容 → 撰写生成 prompt

根据你提供的内容，我会写一个精确的英文 prompt（AI 绘图需要英文描述），包含：

- 尺寸比例（1080×1440，3:4）
- 视觉风格（参考上面 6 种爆款风格选一种）
- 文字内容与位置
- 配色方案
- 禁止元素（不出现真人面孔、不出现品牌 logo 等）

#### Step 2：生成封面图

调用 AI 绘图工具生成封面图。生成后输出：

```
🖼 [封面图片]
⚠️ 发布时必须做的事：
  - 小红书发笔记时勾选「包含 AI 生成内容」
  - 如果不勾选，平台可能限流或处罚
  - 建议：在笔记正文末尾加一句「封面为AI辅助生成」
```

### AI 生成的注意事项

- 文字准确性：AI 生成图片上的文字可能出错，发布前检查拼写
- 风格一致性：如果账号统一风格，AI 生成的可能有偏差，建议用模式 A
- 修改成本：要改一个字就得重新生成，不如模式 A 自己调方便

---

## 设计铁律（两种模式通用）

1. **3色原则**：封面主要颜色 ≤ 3 种
2. **文字可读**：副标题 ≥ 字号 24
3. **信息锚点**：0.3 秒就知道笔记讲什么
4. **合规底线**：禁用极限词（最/第一/国家级）、禁用医疗断言（治愈/根除）、不引导站外引流

## 竞品定位

市面 14 个封面类 Skill 全部是单一模式（要么只生成、要么只顾问）。我们是唯一提供双模式的：

- **模式A**：零门槛设计顾问，给可执行指令，无需安装任何软件
- **模式B**：AI 生成即时出图，标注清晰，合规透明
- 两种模式都覆盖文案 Skill 没解决的最后一环：封面好不好，决定前面的努力有没有白费
