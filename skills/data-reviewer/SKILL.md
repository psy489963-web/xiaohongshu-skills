## 四维分析

不管哪种采集方式，拿到数据后都走同一套分析框架。

### 维度一：流量漏斗（曝光→点击→互动→关注）

```
曝光量 → 点击率 → 互动率 → 涨粉率

例：3200曝光 → 480点击（15%）→ 98互动（20.4%）→ 5涨粉（5.1%）
         ↑               ↑              ↑
      判断点1         判断点2       判断点3
```

| 环节 | 计算公式 | 达标线 | 低于达标线说明 |
|------|---------|:---:|------------|
| 点击率 | 点击量 ÷ 曝光量 | ≥ 5% | 封面+标题不够吸引 |
| 互动率 | 互动数 ÷ 点击量 | ≥ 5% | 内容没留住人/没引导互动 |
| 收藏率 | 收藏数 ÷ 点击量 | ≥ 10% | 干货价值不够 |
| 涨粉率 | 涨粉数 ÷ 点击量 | ≥ 3% | 人设/系列感不够 |
| 评论率 | 评论数 ÷ 点击量 | ≥ 2% | 互动设计缺失 |

### 维度二：流量来源诊断

| 来源 | 健康占比 | 异常解读 |
|------|:---:|---------|
| 推荐 | 40-60% | >70% = 依赖算法，搜索词弱；<30% = 内容不被推荐池接收 |
| 搜索 | 15-30% | <15% = 关键词覆盖不足，标题/正文没埋搜索词 |
| 关注 | 5-15% | >25% = 粉丝粘性高，可加大系列化；<5% = 粉丝不活跃 |
| 个人主页 | 5-10% | 偏高说明主页被反复浏览，置顶笔记选得好 |

### 维度三：内容质量诊断

对每篇笔记做单独诊断：

| 指标 | 计算方式 | 健康值 | 不健康说明 |
|------|---------|:---:|---------|
| 收藏点赞比 | 收藏 ÷ 点赞 | >1.0 | <1.0 = 情绪价值够但干货不足 |
| 分享评论比 | 分享 ÷ 评论 | <2.0 | >3.0 = 社交货币属性强，可复用选题 |
| 评论区质量 | 前5条提问重复度 | — | 同一问题≥3次 = 内容有信息缺口 |

### 维度四：趋势对比

| 对比方式 | 看什么 |
|----------|-------|
| 环比（本周 vs 上周） | 整体趋势上升还是下降 |
| 单篇排名 | 哪篇最好/最差，差多少倍 |
| 发布时间 vs 数据 | 哪个时段发的数据最好 |
| 内容类型 vs 数据 | 干货/避坑/推荐/故事，哪类数据最好 |

## 复盘报告输出格式

```
━━━━━━━━━━━━━━━━━━━━━━
📊 小红书数据复盘报告
复盘周期：近7天 | 生成时间：2026-06-25
━━━━━━━━━━━━━━━━━━━━━━

一、整体概况

| 指标 | 本周 | 上周 | 变化 |
|------|:---:|:---:|:---:|
| 总曝光 | 12,500 | 8,200 | ↑ 52% |
| 总点击 | 1,875 | 1,230 | ↑ 52% |
| 总互动 | 312 | 185 | ↑ 69% |
| 新增粉丝 | 28 | 12 | ↑ 133% |
| 点击率 | 15.0% | 15.0% | → 持平 |
| 互动率 | 16.6% | 15.0% | ↑ 1.6pp |

📌 一句话结论：整体上升，互动率和涨粉提速明显，但点击率没变——流量增长靠曝光量拉动，不是内容质量提升。

二、流量漏斗诊断

曝光 → 点击（15% ✅）→ 互动（16.6% ✅）→ 涨粉（1.5% ⚠️）

✅ 点击率达标（>5%）：封面标题组合有效
✅ 互动率达标（>5%）：内容留住了人
⚠️ 涨粉率偏低（<3%）：读者看了不关注，人设/系列感不够

三、流量来源诊断

| 来源 | 占比 | 状态 |
|------|:---:|:---:|
| 推荐 | 55% | ✅ 健康 |
| 搜索 | 12% | ⚠️ 偏低 |
| 关注 | 25% | ⚠️ 偏高 |
| 个人主页 | 5% | ✅ 正常 |
| 其他 | 3% | — |

📌 搜索流量偏低（<15%）：标题和正文的关键词覆盖不足，用户搜不到你。
📌 关注占比偏高（>25%）：粉丝粘性好，但说明破圈不够，新用户进不来。

四、单篇笔记排名

| # | 标题 | 曝光 | 点击率 | 互动率 | 收藏率 | 诊断 |
|---|------|:---:|:---:|:---:|:---:|------|
| 1 | 实习生存指南 | 5,200 | 18% | 22% | 15% | 🟢 全面优秀 |
| 2 | 租房避坑5条 | 3,800 | 14% | 18% | 12% | 🟢 收藏高 |
| 3 | 通勤穿搭 | 2,100 | 12% | 10% | 5% | 🟡 收藏低 |
| 4 | 谈薪技巧 | 1,400 | 15% | 8% | 3% | 🔴 互动低 |

五、最佳实践 & 最差教训

🟢 最好的一篇：「实习生存指南」
   - 为什么好：搜索流量占35%（标题含"实习""生存指南"高搜词）
   - 可复制点：标题结构「XX生存指南」+ 干货列表型正文

🔴 最差的一篇：「谈薪技巧」
   - 为什么差：推荐给了不对的人（搜索流量仅5%）
   - 问题根因：标题"谈薪"太宽泛，没有场景锁定词

━━━━━━━━━━━━━━━━━━━━━━
```

## 优化建议（联动其他Skill）

复盘报告输出后，紧接着给优化建议。每条建议明确联动哪个 Skill：

```
━━━━━━━━━━━━━━━━━━━━━━
🎯 优化行动清单
━━━━━━━━━━━━━━━━━━━━━━

基于本次复盘，建议接下来做以下调整：

📋 1. 选题方向调整 → 用「选题生成器」
   问题：搜索流量偏低（12%），说明选题不够贴合用户搜索需求
   行动：用选题生成器的「自动选题」模式，重新扫近期高频搜索词
   重点：找搜索量高但笔记少的蓝海选题
   命令：跟我说"帮我自动选题，赛道是XX"

📝 2. 标题关键词优化 → 用「标题生成器」
   问题：点击率达标但搜索流量低，说明标题好看但不含搜索词
   行动：下次写标题时，确保标题包含1-2个用户会搜的关键词
   方法：标题生成器的公式2（痛点+方案）和公式7（数字+清单）都适合埋关键词
   命令：跟我说"帮我写标题，关键词是XX和XX"

✏️ 3. 内容结构优化 → 用「文案优化器」
   问题：涨粉率偏低（1.5%），说明读者看完不想关注
   行动：在正文结尾加"系列预告"或"关注看下期"引导
   方法：用文案优化器检查每篇的结尾，确保有关注引导

🎨 4. 封面一致性 → 用「封面设计顾问」

   方法：用封面设计顾问确定一种固定风格，后续每篇都用
   命令：跟我说"帮我定一个封面风格，赛道是XX"

📅 5. 发布时间调整
   问题：[基于粉丝活跃时段数据的建议]

━━━━━━━━━━━━━━━━━━━━━━
闭环路径：数据复盘 → 优化建议 → 选题生成器 → 标题生成器 → 文案优化器 → 封面设计顾问 → 发布 → 下次复盘

## 周复盘 vs 单篇复盘

### 周复盘（推荐每周一次）

采集近7天全部数据 → 四维分析 → 整体优化建议。

触发词：「帮我做周复盘」「复盘本周数据」

### 单篇复盘（每篇笔记发布后24小时）

聚焦单篇笔记的六项指标 + 流量来源 → 快速诊断 → 针对性优化建议。

触发词：「帮我复盘这篇笔记」「这篇笔记数据怎么样」

单篇复盘输出更简洁：

```
━━━━━━━━━━━━━━━━━━━━━━
📊 单篇笔记复盘：「实习生存指南」
━━━━━━━━━━━━━━━━━━━━━━

六项核心指标：
| 指标 | 数值 | 达标 | 基准线 |
|------|:---:|:---:|:---:|
| 曝光 | 5,200 | — | — |
| 点击率 | 18% | ✅ | ≥5% |

🎯 下一步：
1. 这篇是爆款模板，用「选题生成器」找同类选题复制

| 推荐流量占比 | — | 40-60% | 30-70% | <30%或>70% |

## 红线

- 不要求AI自动登录用户账号（AI做不到，引导用户自己登录）
- 不采集非本人账号数据（只分析用户自己的后台数据）
- 不输出虚假数据（手动模式下缺的数据标注"推断"，不编造）
- 数据分析结论必须有数据支撑，不凭空判断
- 优化建议必须可执行，不输出"努力提升内容质量"这类废话

## 五件套闭环

```
┌─────────────────────────────────────────────────┐
│                                                   │
│   选题生成器 → 标题生成器 → 文案优化器 → 封面顾问  │
│       ↑                                    ↓      │
│       │                                   发布    │
│       │                                    ↓      │
│       └──── 数据复盘器（采集→分析→优化建议）─────┘
│                                                   │
│   说"帮我复盘"启动闭环                            │
│   说"帮我选题"开始新一轮                           │
│                                                   │
└─────────────────────────────────────────────────┘
```

| 环节 | Skill | 解决什么 |
|------|-------|---------|
| 发什么 | 选题生成器 | 找到有流量的选题方向 |
| 标题怎么写 | 标题生成器 | 写出高点击率标题 |
| 正文怎么写 | 文案优化器 | 写出高互动+高收藏正文 |
| 封面怎么做 | 封面设计顾问 | 做出高点击率封面 |
| 效果怎么样 | 数据复盘器 | 用数据验证+指导下一轮 |
