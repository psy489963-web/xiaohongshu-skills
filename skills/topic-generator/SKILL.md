### 模式C 工作流

#### Step 1：确认赛道

用户说「帮我自动选题，赛道是XX」。如果没有说赛道，先问。

#### Step 2：全网扫描

用网络搜索采集近一周相关高频搜索词。搜索策略：

**必须搜的关键词组合：**
- "小红书 [赛道] 热门话题 2026年6月"
- "小红书 [赛道] 热搜词 最近"
- "[赛道] 小红书 爆文 近期"
- "小红书 热搜榜 [赛道] 本周"
- "小红书 [赛道] 趋势 2026"

**采集目标：**
- 高频搜索词及近似词
- 每个词的大致笔记量和互动水平
- 哪些词在上升、哪些在下降
- 评论区高频需求词

#### Step 3：需求验证

对每个候选搜索词做验证：

| 验证维度 | 方法 |
|----------|------|
| 搜索热度 | 相关笔记量级判断 |
| 闲鱼验证 | 搜索"方向+9.9"看是否有付费产品 |
| 互动验证 | 相关笔记是否有大量"求推荐""求链接" |
| 竞争判断 | 笔记数量多但千赞少 = 有需求但内容质量低 = 机会 |

**判断标准：**

| 信号 | 判断 |
|------|------|
| 笔记多 + 千赞多 + 闲鱼有人卖 | 🟢 成熟刚需，做差异化 |
| 笔记少 + 热度涨 + 闲鱼有人卖 | 🟢 蓝海机会，快速占位 |
| 笔记多 + 千赞少 | 🟡 有需求无好内容，可做 |
| 热度涨 + 灰色/争议 | 🔴 不建议（平台合规风险） |
| 无闲鱼 + 无非商业互动 | 🔴 伪需求 |

#### Step 4：输出选题报告

固定格式（见上方示例），必须包含：

1. **高频搜索词表**：热度分级，标注需求类型
2. **上升趋势词**：标注机会判断（蓝海/有风险/竞争大）
3. **需求验证表**：每个候选词的三维验证
4. **选题计划表**：5个具体选题 + 依据 + 预期流量
5. **30天发布配比建议**
6. **第一个行动**：明天发哪篇

### 模式C 注意事项

- 搜索数据有滞后性（基于已发布的内容），不代表未来趋势
- 建议用模式C快速锁定方向，再用模式A做深度调研
- 如果赛道太小众（笔记量 < 1000），模式C可能数据不足，自动降级为模式A

## 选题方向速查

| 方向 | 特征 | 案例 |
|------|------|------|
| 干货攻略 | 信息量大，收藏率高 | "实习生生存指南｜入职前必看10件事" |
| 避坑指南 | 情绪价值+实用，评论多 | "新人租房的5个坑，第3个我踩了" |
| 实用推荐 | 转化强，评论区求链接 | "月薪5k通勤穿搭｜一周不重样" |
| 行业内幕 | 认知差，传播力强 | "HR不会告诉你的谈薪技巧" |
| 亲身经历 | 共情+信任，关注率高 | "入职3个月被裁，我学到了什么" |

## 分析方法

**找对标账号 4 标准**：粉丝0-1万 + 笔记<100篇 + 赞藏评均衡 + 千赞爆文≥3篇

**拆共性（最重要）**：同一博主反复出现的题目结构、封面元素、正文套路——共性才是可复制的流量密码。找到后做升级：她讲8件事，你能不能讲10件？她的图有没有优化空间？

**象限组合法（选题荒）**：人群 × 场景 × 痛点 × 价格 × 季节 → 随意组合。例：学生党 + 宿舍 + 减肥 + 平价 + 秋冬 = "宿舍党秋冬平价瘦腿攻略"。

**评论区淘金**：翻同行热门笔记评论区，用户提问=现成选题——用户主动说出来的需求比调研都准。

## 红线

- 不涉及医疗/金融承诺（"7天治愈""稳赚不赔"等自动拒绝）
- 不输出引流类选题（加微信/私域）
- 禁用词自动替换：最→保姆级 | 第一→首选 | 100%→几乎全部
- 禁止纯蹭热点但与赛道无关的选题（不符合的流量不要）
- 模式C的数据来自公开搜索，仅供参考，强烈建议发布前手动验证
